# Nivel 8: Patrones de Diseno

Soluciones reutilizables para problemas comunes en desarrollo de software.

## Modulos

### 1. Creacionales (01-creacionales)
Singleton, Factory Method, Builder.

### 2. Estructurales (02-estructurales)
Adapter, Decorator, Proxy.

### 3. Comportamiento (03-comportamiento)
Observer, Strategy, Template Method.

### 4. MVC y Repository (04-mvc-repository)
Model-View-Controller, Repository, Inyeccion de Dependencias.

### 5. Clean Code (05-clean-code)
SRP, nombres descriptivos, DRY.

## Como ejecutar

```bash
python scripts/runner.py 8 [modulo] [ejercicio]
```

Ejemplo: `python scripts/runner.py 8 1 1`

## Requisitos

- Java 11+ (para compilar con javac)
