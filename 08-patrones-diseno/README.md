# Nivel 8: Patrones de Diseno

Los patrones de diseno son soluciones reutilizables para problemas comunes en el desarrollo de software. Este nivel cubre los patrones mas importantes y practicas de codigo limpio.

## Modulos

### 01 - Creacionales
Singleton, Factory Method y Builder. Controlan el proceso de creacion de objetos.

### 02 - Estructurales
Adapter, Decorator, Proxy y Facade. Definen relaciones entre objetos y clases.

### 03 - Comportamiento
Observer, Strategy y Template Method. Gestionan algoritmos y responsabilidades.

### 04 - MVC y Repository
Arquitectura MVC, patron Repository e Inyeccion de Dependencias.

### 05 - Clean Code
Convenciones de nombres, funciones pequenas, principio DRY, comentarios vs codigo, refactorizacion.

## Ejecucion

Cada modulo contiene:
- `README.md` -- Teoria y ejemplos
- `Ejercicios.java` -- Practica con 3 ejercicios
- `Soluciones.java` -- Soluciones completas

Compilar y ejecutar:
```
javac Ejercicios.java && java Ejercicios
```
