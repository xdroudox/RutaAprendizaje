> SOLUCION 2: CSS Flexbox y Grid

Soluciones completas para los ejercicios de Flexbox y Grid.

Ejercicio 1: Centrar con Flexbox

SOLUCION: display:flex + justify-content + align-items
.contenedor-flex { display: flex; justify-content: center; align-items: center; }
Explicacion: display: flex activa flexbox en el contenedor. justify-content: center centra horizontalmente (eje principal). align-items: center centra verticalmente (eje cruzado).
Centrado!

Ejercicio 2: Galeria responsiva con Flexbox

SOLUCION: flex-wrap + flex-basis responsivo
.galeria { display: flex; flex-wrap: wrap; gap: 10px; } .galeria .item { flex: 1 1 200px; }
Explicacion: flex-wrap: wrap permite que los items bajen a la siguiente fila. flex: 1 1 200px significa: grow=1 (crecen para llenar espacio), shrink=1 (se encogen si hace falta), basis=200px (tamaño base). gap agrega espacio entre items. Si la pantalla es menor a 200px+gap, los items se apilan en una columna.

Ejercicio 3: Layout completo con Grid

SOLUCION: grid-template-columns/rows + grid-area
.grid-layout { display: grid; grid-template-columns: 200px 1fr 200px; grid-template-rows: 80px 1fr 60px; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer"; gap: 4px; height: 400px; } .grid-layout > div:nth-child(1) { grid-area: header; } .grid-layout > div:nth-child(2) { grid-area: sidebar; } .grid-layout > div:nth-child(3) { grid-area: main; } .grid-layout > div:nth-child(4) { grid-area: aside; } .grid-layout > div:nth-child(5) { grid-area: footer; }
Explicacion: grid-template-columns: 200px 1fr 200px crea 3 columnas: sidebar fija, main flexible, aside fijo. grid-template-rows: 80px 1fr 60px header 80px, contenido flexible, footer 60px. grid-template-areas nombra las regiones del layout. Cada hijo usa grid-area para asignarse a una region. El orden en el HTML no importa porque la posicion la define grid-area.
