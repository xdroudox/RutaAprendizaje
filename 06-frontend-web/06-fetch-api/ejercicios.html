Completa las funciones en la seccion "TU CODIGO AQUI" para consumir la API de JSONPlaceholder.

Ejercicio 1: GET - Obtener posts

Cada post debe mostrar: titulo (en negrita) y cuerpo del texto.

PISTA: fetch('https://jsonplaceholder.typicode.com/posts').then(r => r.json()).then(data => data.slice(0, 5)). Usa .map() para crear elementos HTML.
