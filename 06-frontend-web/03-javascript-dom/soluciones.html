Ejercicio 1: Manipular texto y estilos

SOLUCION
function ejercicio1() { var parrafo = document.getElementById('parrafo-ej1'); parrafo.textContent = 'Texto cambiado exitosamente!'; parrafo.classList.add('destacado'); }
Explicacion: document.getElementById('parrafo-ej1') obtiene el elemento por su ID. .textContent cambia el texto interno. .classList.add('destacado') agrega la clase CSS 'destacado' al elemento.

Ejercicio 2: Crear y agregar elementos

SOLUCION
var contadorEj2 = 0; function ejercicio2() { var contenedor = document.getElementById('contenedor-ej2-sol'); contadorEj2++; var nuevoDiv = document.createElement('div'); nuevoDiv.textContent = 'Elemento #' + contadorEj2; nuevoDiv.classList.add('card'); contenedor.appendChild(nuevoDiv); } function limpiarEj2() { document.getElementById('contenedor-ej2-sol').innerHTML = ''; contadorEj2 = 0; }
Explicacion: document.createElement('div') crea un nuevo elemento div en memoria (no en la pagina). Se le asigna texto con textContent y clase con classList.add. appendChild lo agrega al DOM dentro del contenedor. innerHTML = '' limpia todo el contenido del contenedor.

Ejercicio 3: Lista interactiva (To-Do List)

SOLUCION
function agregarTarea() { var input = document.getElementById('input-tarea-sol'); var lista = document.getElementById('lista-tareas-sol'); var texto = input.value.trim(); if (texto === '') return; var li = document.createElement('li'); var span = document.createElement('span'); span.textContent = texto; li.appendChild(span); var btnEliminar = document.createElement('button'); btnEliminar.textContent = 'Eliminar'; btnEliminar.classList.add('btn-eliminar'); btnEliminar.onclick = function(e) { e.stopPropagation(); li.remove(); }; li.appendChild(btnEliminar); li.onclick = function() { li.classList.toggle('completada'); }; lista.appendChild(li); input.value = ''; }
Explicacion: Se crea un li con dos hijos: un span para el texto y un button para eliminar. e.stopPropagation() evita que el clic en el boton dispare el clic del li. classList.toggle('completada') agrega o quita la clase que tacha el texto. li.remove() elimina el elemento del DOM.
