> EJERCICIO 3: JavaScript DOM

Completa las funciones JavaScript en la seccion "TU CODIGO AQUI". Recarga la pagina y prueba los botones.

Ejercicio 1: Manipular texto y estilos

Al hacer clic en el boton, debe cambiar el texto del parrafo y aplicar la clase "destacado".

PISTA: Usa document.getElementById(), .textContent, .classList.add()

Texto original - haz clic en el boton para cambiarme

Ejercicio 2: Crear y agregar elementos

Al hacer clic en "Agregar elemento", se debe crear un nuevo div con la clase "card" y texto "Elemento #N" (donde N es el numero secuencial) y agregarlo al contenedor.

El boton "Limpiar" debe eliminar todos los elementos agregados.

PISTA: document.createElement(), .textContent, .classList.add(), .appendChild(). Para limpiar, usa innerHTML = '' en el contenedor.

Ejercicio 3: Lista interactiva

Crea una lista de tareas (to-do list) completa:

  • Al hacer clic en "Agregar", lee el valor del input y crea un nuevo elemento li en la lista
  • Cada li debe tener un boton "Eliminar" que al hacer clic elimine ese li
  • Al hacer clic en un li (no en el boton), alterna la clase "completada" (tachar el texto)
PISTA: Usa createElement, textContent, appendChild, addEventListener. Para el boton eliminar, usa event.target.parentElement.remove().
