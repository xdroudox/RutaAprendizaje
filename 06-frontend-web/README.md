## Temas

| # | Modulo | Descripcion |
|---|--------|-------------|
| 1 | **HTML Semantico** | header, nav, main, section, article, aside, footer, accesibilidad |
| 2 | **CSS Flexbox y Grid** | display flex, justify/align items, grid-template, areas |
| 3 | **JavaScript DOM** | document.getElementById, querySelector, createElement, textContent, innerHTML |
| 4 | **Eventos y Asincronia** | addEventListener, callbacks, Promises, async/await, setTimeout |

## Al completar este nivel podras:

- Construir paginas web semanticas y accesibles
- Maquetar layouts modernos con Flexbox y Grid
- Manipular el DOM con JavaScript
- Manejar eventos y codigo asincrono
- Consumir APIs desde el navegador
- Entender las bases de la seguridad web

## Como usar este nivel

Cada modulo contiene:

```
  modulo/
  +-- README.md        -> Teoria + ejercicios
  +-- ejercicios.html  -> Abre en navegador, edita el codigo
  +-- soluciones.html  -> Solucion explicada
  +-- ejercicios.py    -> Instrucciones desde terminal
  +-- soluciones.py    -> Soluciones desde terminal
```

### Usando el runner (recomendado)

Desde la raiz del proyecto:

```
python scripts/runner.py 6        -> Lista modulos
python scripts/runner.py 6 1      -> Lista ejercicios del modulo
python scripts/runner.py 6 1 1    -> Ejecuta ejercicio 1
python scripts/runner.py 6 1 1 -s -> Ver solucion
```

### Manualmente

1. Abre README.md y lee la teoria
2. Abre ejercicios.html en tu navegador (doble clic)
3. Edita el archivo HTML para resolver los ejercicios
4. Recarga el navegador para ver los cambios
5. Revisa soluciones.html si te atascas

El modulo 05-navegadores es solo teoria (no tiene ejercicios practicos).
