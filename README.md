## 🎯 ¿Qué es esto?

Un **roadmap RPG interactivo** para aprender desarrollo full-stack desde cero hasta arquitectura de software. No es solo teoría aburrida: **cada tema tiene ejercicios prácticos que puedes ejecutar desde la terminal**.

## 🗺️ Los 11 Niveles

```
N1  ──→  N2  ──→  N3  ──→  N4  ──→  N5  ──→  N6
 │                            │        │
 └──────→  N7  ──→  N8  ──→  N9  ──→  N10 ──→  N11
```

| # | Nivel | Lenguaje | ¿Qué aprenderás? |

## 🎮 Cómo usar esto

Cada submódulo tiene 3 archivos:

```
📁 01-tema/
├── 📖 README.md        → Teoría + ejemplos + enlaces a ejercicios
├── 🎯 ejercicios.py    → Ejercicios interactivos (ejecuta desde terminal)
└── 🔍 soluciones.py    → Soluciones explicadas
```

### Flujo de aprendizaje:

```
1. Abres README.md      → Lees teoría con ejemplos
2. Ejecutas ejercicio   → python ejercicios.py 1
3. Escribes tu código   → En el espacio designado del archivo
4. Pruebas solución     → python ejercicios.py 1 (de nuevo)
5. ¿Atascado?           → python ejercicios.py 1 -p (pista)
6. Ves solución         → python soluciones.py 1
7. 🎉 Siguiente tema!
```

## ⚙️ Requisitos

- **Python 3.8+** → `python --version`
- **Java 17+** → `java -version` y `javac -version`
- **SQLite3** (viene con Python)
- **Git** → `git --version`

> 💡 Ejecuta `python scripts/check-requirements.py` para verificar tu setup
