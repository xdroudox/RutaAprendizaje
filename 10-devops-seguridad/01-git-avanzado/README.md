# 01 - Git Avanzado

## Conceptos clave

- **Branching**: ramas para desarrollo paralelo.
- **Merge**: integrar cambios de una rama a otra.
- **Rebase**: reubicar commits sobre otra base.
- **Conflictos**: cuando dos cambios modifican la misma linea.
- **Stash**: guardar cambios temporalmente.

## Ejercicios

1. **Init, add y commit** - Crea un repositorio y realiza tu primer commit.
2. **Ramas y merge** - Trabaja con ramas y fusiona cambios.
3. **Resolver conflicto** - Simula y resuelve un conflicto de merge.
