# Nivel 10: DevOps y Seguridad

Bienvenido al nivel final de la Ruta de Aprendizaje. Este nivel combina dos pilares
fundamentales del desarrollo moderno: las practicas DevOps y la seguridad informatica.

## Objetivos

- Dominar Git avanzado: branching, merging y resolucion de conflictos.
- Contenerizar aplicaciones con Docker y orquestar con Docker Compose.
- Implementar pipelines de CI/CD con GitHub Actions.
- Conocer las vulnerabilidades mas criticas del OWASP Top 10.
- Aplicar tecnicas de cifrado y hashing en aplicaciones Python.
- Proteger aplicaciones web contra ataques comunes.

| Modulo | Tema | Tipo |

## Ejecucion

Cada modulo tiene ejercicios que se ejecutan con:

```
python scripts/runner.py 10 [modulo] [ejercicio]
```

Ejemplo: `python scripts/runner.py 10 01 1` ejecuta el ejercicio 1 de Git Avanzado.
