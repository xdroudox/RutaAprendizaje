## Que aprenderas en este nivel

Las **estructuras de datos** son formas de organizar y almacenar informacion para que podamos accederla y modificarla eficientemente. Este nivel cubre desde las estructuras mas simples (arrays, listas) hasta las mas complejas (arboles, grafos), junto con los algoritmos que las manipulan.

Cada tema sigue la misma estructura:

```
📖 TEORIA → Explicacion con ejemplos linea por linea
📖 GLOSARIO → Terminos del tema definidos
🔄 COMPARATIVA → Mismo concepto en Python / Java / JavaScript
📝 EJEMPLO GUIADO → Problema resuelto paso a paso
🎯 EJERCICIOS → 3 niveles: 🟢 Basico, 🟡 Intermedio, 🔴 Avanzado
```

## Temas

| # | Tema | Que aprenderas |
|---|------|----------------|
| 1 | **Complejidad Algoritmica** | Big O, O(1), O(n), O(n^2), O(log n), O(n log n) |
| 2 | **Arrays y Listas Enlazadas** | Arrays estaticos/dinamicos, singly/doubly linked lists |
| 3 | **Pilas y Colas** | Stack LIFO, Queue FIFO, deque, aplicaciones reales |
| 4 | **Arboles** | Binary trees, BST, recorridos: inorder, preorder, postorder |
| 5 | **Grafos** | Dirigidos/no dirigidos, adjacency list/matrix, DFS, BFS |
| 6 | **Ordenamiento** | Bubble, selection, insertion, quick, merge sort |
| 7 | **Busqueda** | Lineal, binaria, busqueda por hash, patrones |
| 8 | **Recursion** | Recursion directa/indirecta, tail recursion, backtracking |

## Progresion

```
T1 (Complejidad) → T2 (Arrays/Listas) → T3 (Pilas/Colas) → T4 (Arboles)
                             │                                    │
                  T6 (Ordenamiento) ←────────────────────── T5 (Grafos)
                             │
                  T7 (Busqueda) → T8 (Recursion)
```
