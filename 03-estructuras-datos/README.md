# Nivel 3: Estructuras de Datos

Estructuras de datos fundamentales: arrays, listas enlazadas, pilas, colas, arboles, grafos, algoritmos de ordenamiento y busqueda, recursion.

## Temas

| # | Tema | Descripcion |
|---|------|-------------|
| 1 | **Complejidad Algoritmica** | Big O: O(1), O(n), O(n^2), O(log n), O(n log n) |
| 2 | **Arrays y Listas Enlazadas** | Arrays vs linked lists, singly/doubly, operaciones |
| 3 | **Pilas y Colas** | Stack LIFO, Queue FIFO, deque, implementaciones |
| 4 | **Arboles** | Binary trees, BST, traversals: inorder/preorder/postorder |
| 5 | **Grafos** | Dirigidos/no dirigidos, adjacency list/matrix, DFS, BFS |
| 6 | **Ordenamiento** | Bubble, selection, insertion, quick, merge sort |
| 7 | **Busqueda** | Linear search, binary search, hash table search |
| 8 | **Recursion** | Funciones recursivas, factorial, fibonacci, backtracking basics |

## Como empezar

```bash
# Comienza con el primer tema
code 03-estructuras-datos/01-complejidad-algoritmica/README.md

# Ejecuta ejercicios
python 03-estructuras-datos/01-complejidad-algoritmica/ejercicios.py
```

## Al completar este nivel podras

- Analizar la eficiencia de algoritmos usando Big O
- Implementar estructuras de datos desde cero
- Recorrer y manipular arboles y grafos
- Ordenar y buscar datos eficientemente
- Resolver problemas con recursion
