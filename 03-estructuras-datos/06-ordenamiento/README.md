# Ordenamiento

Algoritmos para ordenar elementos de una coleccion.

- **Bubble Sort**: O(n^2) - Compara e intercambia pares adyacentes
- **Selection Sort**: O(n^2) - Selecciona el minimo en cada pasada
- **Insertion Sort**: O(n^2) - Inserta cada elemento en su posicion
- **Merge Sort**: O(n log n) - Divide y conquista
- **Quick Sort**: O(n log n) promedio - Particiona y ordena
