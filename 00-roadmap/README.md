## 📊 Progresión de Habilidades

| Nivel | Título | XP | Habilidades Clave |
|-------|--------|----|--------------------|
| **1** | Fundamentos Python | 100 XP | Algoritmos, variables, control flujo, funciones, colecciones |
| **2** | POO + SOLID Java | 200 XP | Clases, herencia, polimorfismo, interfaces, SOLID |
| **3** | Estructuras Datos | 300 XP | Big O, listas, árboles, grafos, sort, search |
| **4** | Bases de Datos | 200 XP | SQL, normalización, joins, transacciones, NoSQL |
| **5** | Backend + APIs | 400 XP | HTTP, REST, JWT, JSON, redes, middleware |
| **6** | Frontend Web | 300 XP | HTML, CSS, DOM, eventos, fetch, browsers |
| **7** | Testing | 200 XP | Unit tests, TDD, mocking, integración |
| **8** | Patrones Diseño | 300 XP | GoF patterns, MVC, Clean Code |
| **9** | Arquitecturas | 400 XP | Capas, hexagonal, microservicios, DDD, CQRS |
| **10** | DevOps + Seguridad | 300 XP | Git, Docker, CI/CD, OWASP, cifrado |
| **11** | 🏆 Proyecto Final | 1000 XP | Full-stack app completa |

## 📚 Convenciones del Repositorio

- Cada tema tiene 3 archivos: `README.md`, `ejercicios.*`, `soluciones.*`
- Ejecuta ejercicios con: `python ejercicios.py N` o `java Ejercicios N`
- Pista: `python ejercicios.py N -p`
- Solución: `python soluciones.py N`
- Marca tu progreso editando el `skills-tree.md`
