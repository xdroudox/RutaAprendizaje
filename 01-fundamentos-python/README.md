## Que aprenderas en este nivel

Este nivel cubre desde CERO los fundamentos de la programacion usando Python. No necesitas experiencia previa.

Cada tema sigue la misma estructura:

```
📖 TEORIA → Explicacion con ejemplos linea por linea
📖 GLOSARIO → Terminos del tema definidos
🔄 COMPARATIVA → Mismo concepto en Python / Java / JavaScript
📝 EJEMPLO GUIADO → Problema resuelto paso a paso
🎯 EJERCICIOS → 3 niveles: 🟢 Basico, 🟡 Intermedio, 🔴 Avanzado
```

## Temas

| # | Tema | Que aprenderas |
|---|------|----------------|
| 1 | **Logica de Programacion** | Algoritmos, pseudocodigo, resolucion de problemas |
| 2 | **Variables y Tipos de Datos** | int, float, string, bool, type(), conversion |
| 3 | **Operadores y Expresiones** | Aritmeticos, comparacion, logicos, asignacion |
| 4 | **Control de Flujo** | if/elif/else, for, while, break, continue |
| 5 | **Funciones** | def, parametros, retorno, scope, lambda |
| 6 | **Strings y Manejo de Texto** | slicing, metodos, formato, f-strings |
| 7 | **Colecciones** | list, tuple, dict, set, comprehensions |

## Progresion

Los temas estan en orden. NO te saltes ninguno:

```
T1 (Logica) → T2 (Variables) → T3 (Operadores) → T4 (Control Flujo)
                                                      │
                                        T5 (Funciones) ←┘
                                              │
                                   T6 (Strings) ──→ T7 (Colecciones)
```

Cada tema asume que ya entiendes los anteriores.

## Al completar este nivel podras:

- Escribir programas basicos en Python
- Resolver problemas usando algoritmos simples
- Manejar datos con variables y colecciones
- Crear funciones reutilizables
- Entender como otros lenguajes hacen lo mismo (Java, JavaScript)
