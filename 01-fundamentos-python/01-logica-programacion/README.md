# Logica de Programacion

Un **algoritmo** es una secuencia de pasos ordenados para resolver un problema. Como una receta de cocina:

```
1. Conseguir ingredientes
2. Mezclar ingredientes
3. Cocinar
4. Servir
```

En programacion, los algoritmos se escriben en un lenguaje que la computadora entienda (Python, Java, etc).

## Caracteristicas de un buen algoritmo

- **Preciso**: Cada paso esta claramente definido
- **Finito**: Tiene un final claro
- **Determinista**: Misma entrada = misma salida

## Ejemplo en Python

```python
# Algoritmo: saludar al usuario
nombre = input("Como te llamas? ")
print(f"Hola, {nombre}!")
```

## Ejercicios

### Ejercicio 1: Preparar un sandwich
Escribe un algoritmo (en codigo) que muestre los pasos para preparar un sandwich.

**Ejecuta:** `python scripts/runner.py 1 1 1`

### Ejercicio 2: Par o impar
Pide un numero al usuario y determina si es par o impar.

**Ejecuta:** `python scripts/runner.py 1 1 2`

### Ejercicio 3: Mayor de 3 numeros
Pide 3 numeros y muestra el mayor.

**Ejecuta:** `python scripts/runner.py 1 1 3`

```bash

python scripts/runner.py 1 1 2 -s
```
